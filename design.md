# Design Philosophy

The interface should feel:

* Premium
* Intelligent
* Modern
* Clean
* Fast
* Career-focused

The platform should look like a professional SaaS product, not a college project.

Avoid:

* Bright gradients everywhere
* Cartoon illustrations
* Generic admin dashboard designs
* Bootstrap-like UI

Focus on:

* Clean layouts
* Strong typography
* Elegant motion
* High information density
* Professional visual hierarchy
